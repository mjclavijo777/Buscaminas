Este proyecto es una implementación del juego BuscaMinas en Java, donde el jugador debe descubrir todas las casillas seguras sin detonar una mina.
El programa utiliza un tablero representado como una matriz, en el que cada celda puede contener una mina o un número que indica cuántas minas hay alrededor. El jugador realiza jugadas seleccionando posiciones del tablero, y el sistema valida cada movimiento, mostrando si la celda es segura o si contiene una mina.
El juego finaliza cuando:

 El jugador pisa una mina (derrota)
Se descubren todas las casillas seguras (victoria)

 Objetivo
Practicar el uso de matrices, validación de datos, lógica de juegos y programación orientada a objetos en Java, creando una versión funcional del clásico Buscaminas en consola.
